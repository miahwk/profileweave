# 交付价值与验收清单

## 1. 价值主张

第一版不以“开关数量”验收，而以用户可完成的工作验收。

### 价值 A：五分钟创建一个可复用的隔离环境

用户可以创建 profile，保存名称、启动页、语言/时区/屏幕意图和代理；关闭应用再打开后仍存在。

验收：

- 创建必填项清晰，常见桌面模板可直接使用。
- profile 元数据原子持久化。
- 每个 profile 的 cookies/localStorage/cache 位于独立目录。
- 复制 profile 会生成新 ID 和新数据目录，不复制运行进程。

### 价值 B：启动前知道配置是否自洽、是否真的生效

用户能看到健康分、错误/警告，以及每个能力是已应用、部分应用还是未支持。

验收：

- UA 与 OS 冲突、非法时区、非法代理、异常屏幕等被发现。
- `unsupported` 字段不会用“已保护”或“已伪装”表述。
- 存在错误时拒绝启动；警告保留并清楚展示。
- 默认使用原生 UA，减少 Client Hints/引擎版本矛盾。

### 价值 C：一键启动和停止本地浏览器

用户不需要手写命令行，就能用找到的 Chrome/Edge/Brave/Chromium 启动独立 profile。

验收：

- 自动发现常见浏览器或选择自定义可执行文件。
- 启动命令不经过 shell，参数来自已验证的配置。
- 同一个 profile 不能重复启动。
- UI 展示 PID、开始时间和状态，并能停止由本应用启动的进程。

### 价值 D：默认不扩大本机攻击面

验收：

- 服务默认只监听 loopback。
- 跨 Origin 的写请求被拒绝。
- 不持久化代理密码，不暴露远程 CDP。
- 不能通过 profile ID 或浏览器路径实现目录穿越/参数注入。

### 价值 E：代码可以持续演进

验收：

- Vue/Go 分层符合 RFC 的 DDD 依赖方向。
- 前端源文件目标不超过 400 行，Go 源文件目标不超过 300 行。
- 领域规则、仓储、进程参数、HTTP 与前端关键状态均有测试。
- `go test ./...`、`go vet ./...`、前端测试和 production build 通过。

### 价值 F：误删可恢复，敏感数据可明确清理

验收：

- 普通删除将 Profile 配置和浏览器数据移入本地回收站，不立即销毁。
- 回收项明确标记浏览器数据是否仍可恢复。
- 恢复元数据失败时回滚浏览器数据移动，避免半恢复状态。
- 永久删除有二次确认，同时清理 tombstone 和对应浏览器数据。

### 价值 G：能解释“当前到底运行了什么”

用户可从管理台运行本机诊断，看到 Runtime Provider 的来源、许可、版本管理方式、发现/可用浏览器数量、活动会话和能力边界；也可打开 `/self-check` 查看浏览器实际暴露的 UA、平台、语言、时区、屏幕和硬件提示值。

验收：

- `GET /api/v1/doctor` 返回结构化报告，不回显底层错误或浏览器绝对路径。
- `/api/v1/capabilities` 的能力项由当前 Provider 元数据生成，不维护重复静态清单。
- 未发现浏览器时返回可操作建议，不把系统标记为健康。
- 自检页明确声明只显示可观察值，不承诺绕过检测或站点风控。

### 价值 H：Windows 用户可以安装、退出、升级和卸载

用户不需要配置 Go/Node 工具链或手动维护前端目录，就能以普通用户权限安装并从开始
菜单启动管理台。应用文件与 Profile 数据分离，升级/卸载不会把本地会话数据当作程序
文件误删。

验收：

- amd64/arm64 setup 使用固定 AppId 和 `%LOCALAPPDATA%\Programs\ProfileWeave`。
- 安装器拒绝 Windows 10 build 17763 / Windows Server 2019 之前的不受支持系统。
- 启动成功监听 loopback 后才打开管理台，重复启动验证身份并复用现有实例。
- 管理台和开始菜单均可触发统一退出，受管浏览器会话在退出前停止。
- 安装器静默安装/启动/退出/卸载 smoke 通过，卸载后数据 sentinel 保留。
- 未配置 Authenticode 时明确标为 unsigned preview，并提供校验和与 provenance。

## 3. 明确不计入本次完成度

- “通过某个检测网站”不是验收项。
- 批量账号、自动登录、验证码解决和目标站点规避不是验收项。
- canvas/WebGL/audio/字体的内核级伪装不是 MVP。
- 云同步、团队权限、远程集群和代理供应商集成不是 MVP。
